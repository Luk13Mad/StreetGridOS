# Hardware Specifications

## Inverter Requirements
- Grid-Forming Inverter (Anchor Node): Capable of Black Start and Frequency Shift Power Control.
- Grid-Tie Inverter (Participant Nodes): UL 1741 SA (Rule 21) compliant.

## Edge Controller (StreetGrid Node)
- **Compute**: Raspberry Pi Zero 2W.
- **Radio**: SX126x LoRa HAT.
- **Sensing**: Non-invasive CT Clamps.
